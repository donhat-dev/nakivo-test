#. Mark the partners that should be allowed to act as resellers by enabling
   the ``Is Reseller`` flag on their ``res.partner`` record.
#. Link customers, opportunities, sales orders, and invoices to a reseller
   through the ``Reseller Partner`` field.
#. Add the portal user that represents the reseller to the
   ``Reseller Portal User`` group.
#. Log in with that user and navigate to ``/my/reseller-portal``.
