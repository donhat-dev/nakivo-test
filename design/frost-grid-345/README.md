# frost-grid-345

## Persona

`frost-grid-345` is a modern SaaS partner portal direction for reseller operations.
It uses a white-first workspace, light-blue command color, mono typography, sharp
edges, and restrained background blur.

## Style Summary

- Light mode only
- `border-radius: 0` across the interface
- `Geist Mono` for interface text and `Space Grotesk` for large headings
- White canvas with zinc borders and one light-blue primary color
- GSAP scroll pinning and scrubbed text reveal for motion
- Static mock data only; no Odoo controller or authenticated route calls

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS v3
- TanStack Query
- GSAP + `@gsap/react`
- `@phosphor-icons/react`

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
