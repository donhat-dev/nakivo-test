# mono-capsule-764

## Persona

CODENAME: `mono-capsule-764`

This direction treats a SaaS partner portal as a quiet command capsule: restrained monochrome surfaces, large asymmetric product framing, soft background blur, and scroll motion that clarifies reseller workflows.

## Aesthetic Vector

- Composition Variance 7/9: asymmetric hero, offset product shell, and dense but controlled portal modules.
- Motion Character 6/9: GSAP scroll reveal, image scale/fade, and gentle ambient movement.
- Visual Density 4/9: enough operational data for partner portal context without cockpit-level compression.

## Stack

- React + Vite
- TypeScript
- Tailwind CSS v3
- GSAP + `@gsap/react`
- Phosphor Icons

## Commands

```bash
npm install
npm run build
npm run lint
npm run dev
```

## Intentional Constraints

- Monochrome warm-neutral palette.
- Light background with subtle blurred ambient layers.
- No production Odoo routes or addon assets are touched.
