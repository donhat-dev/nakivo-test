# clean-signal-455

Persona codename: `clean-signal-455`

Style summary: a sharp, technical reseller portal concept using white and cool gray surfaces, electric blue for primary flow, and `#21B799` green for live or healthy state. The demo keeps the persona's zero-radius geometry and uses Space Grotesk across display and body type.

Stack:

- React + Vite
- TypeScript
- Tailwind CSS v3
- GSAP + `@gsap/react`
- TanStack Query
- Phosphor Icons

Commands:

```bash
npm install
npm run dev
npm run build
npm run lint
```

Routes:

- `/` product landing and signal console narrative
- `/login` reseller portal sign-in concept

Intentional constraints:

- `border-radius: 0px` across the interface
- light mode only
- no Odoo controllers or authenticated production routes
- mock-data driven reseller operations content
- GSAP used for reveal, pinning, image scale, and scrubbed text
