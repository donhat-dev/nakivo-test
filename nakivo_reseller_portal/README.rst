Authenticated reseller portal built on Odoo 19 Community Edition.

Provides:

- a portal page at ``/my/reseller-portal``
- a reseller-scoped dashboard exposing opportunities, quotations, sales
  orders, invoices, and customers
- authenticated JSON endpoints to read, create, and delete
  reseller-owned opportunities
- backend-enforced data isolation per reseller through ACLs, record
  rules, and controller-level ownership checks

Usage
=====

1. Mark the partners that should be allowed to act as resellers by
   enabling the ``Is Reseller`` flag on their ``res.partner`` record.
2. Link customers, opportunities, sales orders, and invoices to a
   reseller through the ``Reseller Partner`` field.
3. Add the portal user that represents the reseller to the
   ``Reseller Portal User`` group.
4. Log in with that user and navigate to ``/my/reseller-portal``.
